## rules.i存储结构

<词类名称>变形名称:规则字符串

示例：

<AR>第一人称现在时一般体陈述式主动态:+ie

+a添加，-a去掉

如果要更复杂的条件：

辅音+ex,元音+x

可以使用正则表达式（外面用[]包起来）：

[[aeiou]$]+x,+ex

表示，对于aeiou结尾的词，+x，否则加ex。

逗号隔开，意味着判断条件不一样。
